Starting An Exam

Instructions

To start an exam, fill out the course number; ignore the section number and use the format CISCXXX.

Fill out the exam name (i.e. Midterm I, Final, Quiz) and the port number. The port number will specify the location of your Moodle course; you would use the url 64.225.15.171:[port number] to share the assessment with your students.

If you would like the assignment to be timed, check the box and specify the duration in minutes. The timer will start once the course has been created.

Next, upload the Moodle course which contains the assignment as a ZIP file.

Finally, click the Create Exam button and your exam will be live once the Moodle course has been uploaded.
