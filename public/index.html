About

This will be utilized by University of Delaware staff that would like to use Moodle for testing purposes. Here, they will be able to upload a Moodle course containing the assignment. They can either delete the course manually (which will end the assignment) or set a timer that will delete the course, ending the assignment.
×

Instructions

To start an exam, click the "Create Exam" button. To end an exam, find your desired exam and click the “End Exam” button. By doing so, the Moodle course where your students completed the assessment will be downloaded as a zip file, and then will be destroyed. That zip file will automatically start downloading to your local machine.

Instructions

To start an exam,fill out the course number; ignore the section number and use the format CISCXXX. Fill out the exam name (i.e. Midterm I, Final, Quiz) and the port number. The port number will specify the location of your Moodle course; you would use the url 64.225.15.171:[port number] to share the assessment with your students. If you would like the assignment to be timed, check the box and specify the duration in minutes. The timer will start once the course has been created. Next, upload the Moodle course which contains the assignment as a ZIP file. Finally, click the Create Exam button and your exam will be live once the Moodle course has been uploaded.
